## 📌 Deskripsi Repository

Repository ini digunakan untuk menyimpan dan mengelola seluruh tugas mata kuliah **Sistem Informasi Web**.  
Project yang dibuat berupa pengembangan website secara bertahap, dimulai dari website skincare hingga website penjualan risol.

---

## 📂 Struktur Tugas

### [Week 1](./Week%20satu%20fix/Week1%20Tugas)  
Pada week ini dibuat project awal berupa **website skincare sederhana**.  

Fitur yang ada:
- Tampilan produk skincare
- Struktur dasar HTML
- Styling sederhana menggunakan CSS

---

###  [Week 2](./Week%202/TA_week2)  
Pada week ini dibuat website penjualan risol dengan fitur interaksi sederhana.  

Fitur yang ada:
- Menampilkan daftar produk risol
- Tombol **Beli**
- Jika tombol beli ditekan, maka stok barang akan **berkurang otomatis**

---

###  [Week 3](./Week%203/TA_week2)  
Pada week ini dilakukan pengembangan fitur interaksi pada website risol.  

Fitur tambahan:
- Tombol **Wishlist**
- Fitur **Detail Produk**
- Interaksi user lebih dinamis menggunakan JavaScript

---

## Tentang Project

Project ini merupakan latihan pembuatan website berbasis **HTML, CSS, dan JavaScript**, yang dikembangkan secara bertahap setiap minggu.

---

## Catatan

- Setiap week merupakan lanjutan/pengembangan dari sebelumnya  
- Project dibuat untuk keperluan pembelajaran
