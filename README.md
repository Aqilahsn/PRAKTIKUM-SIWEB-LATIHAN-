Struktur Tugas

Week 1
Pada folder [Week1](./Week1) dibuat project awal berupa website skincare sederhana.

Fitur yang ada:

* Tampilan produk skincare
* Struktur dasar HTML
* Styling sederhana menggunakan CSS

Week 2
Pada folder [Week2](./Week2) dibuat website penjualan risol dengan fitur interaksi sederhana.

Fitur yang ada:

* Menampilkan daftar produk risol
* Tombol Beli
* Jika tombol beli ditekan, maka stok barang akan berkurang otomatis

Week 3
Pada folder [Week3](./Week3) dilakukan pengembangan fitur interaksi pada website risol.

Fitur tambahan:

* Tombol Wishlist
* Fitur Detail Produk
* Interaksi user lebih dinamis menggunakan JavaScript

Week 4
Pada folder [Week4](./Week4) dilakukan migrasi project dari native (HTML, CSS, JavaScript) ke Laravel.

Perubahan yang dilakukan:

* Membuat project menggunakan Laravel
* Mengubah HTML menjadi Blade Template
* Mengatur routing menggunakan web.php
* Menyusun struktur folder menggunakan konsep MVC
* Memindahkan asset ke folder public

Tentang Project
Project ini merupakan latihan pembuatan website berbasis HTML, CSS, dan JavaScript yang dikembangkan secara bertahap setiap minggu, kemudian dilanjutkan dengan penggunaan framework Laravel.

Catatan

* Setiap folder Week merupakan lanjutan dari sebelumnya
* Project dibuat untuk keperluan pembelajaran
* Struktur folder: Week1, Week2, Week3, Week4
